# python-web-pants-bottle-hello-world

## Description
POC for bottle web framework with bazel.

## Tech stack
- python
  - bottle
- pants

## Docker stack
- pantsbuild/centos7:latest

## To run
`sudo ./install.sh -u`
- [Available at](http://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
